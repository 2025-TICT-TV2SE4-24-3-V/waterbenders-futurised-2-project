# Project Team WaterBenders

Zie de [README.md](docs/README.md) in folder `docs/` voor meer informatie en instructies over het uitvoeren van het project.

## Introductie

Deze repository bevat al ons gemaakte werk voor het project in opdracht van de HU en Futurised.

## Opdracht

Simulatie van blus robot Flip met toevoeging van autonome kenmerken, zoals path-finding, objectherkenning en 3D-mapping.
